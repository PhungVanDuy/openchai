# OpenChai

### Installation:
```bash
git clone https://github.com/OpenAccess-AI-Collective/axolotl
cd axolotl

pip3 install -e .[flash-attn]
pip3 install -U git+https://github.com/huggingface/peft.git
```

### Training:
Example:

```bash
openchai train --config_path examples/llama2/7b_chatml.yaml
```
